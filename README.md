# Microsoft Project (microsoft-project)
Microsoft Project is a project management software product developed and sold by Microsoft. It is designed to assist project managers in developing plans, assigning resources to tasks, tracking progress, managing budgets, and analyzing workloads.

**URL:** [Visit APIs.json URL](https://www.microsoft.com/en-us/microsoft-365/project/project-management-software)
